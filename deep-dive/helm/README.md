## Helm



---
### List Repo
```sh
helm repo list
```
---
### List package installed
```sh
helm list
```
---
#### Add Repo
```sh
helm repo add <repo-name> <repo-url>
```
### Install  and uninstall a chart
```sh
helm install <name> <repo-name>/<package>
```
```sh
helm uninstall <name>
```
